Sistem Presensi Mandiri (SIPREMAN) adalah sebuah website yang digunakan untuk melakukan presensi secara mandiri oleh mahasiswa dengan menggunakan face recognition dan image classification.

Berikut adalah beberapa menu yang ada di website SIPREMAN.

Registrasi Wajah
Digunakan untuk mendaftarkan wajah mahasiswa sebelum melakukan presensi.
Masuk Kelas
Digunakan untuk melakukan presensi pada suatu kelas.
Table Kehadiran
Digunakan untuk melihat detail dan tabel kehadiran dari suatu kelas.
Generate Token
Digunakan untuk membuat token kelas yang akan digunakan sebagai akses presensi oleh mahasiswa
